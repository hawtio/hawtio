Demos

  • various demos of using hawtio, ActiveMQ and Camel inside JBoss Fuse by James Strachan. JBoss Fuse 6.1 or later uses hawtio for its management console; so all these demos are really hawtio demos (in terms of the console).
  • Demo of Fuse 6.1 with Apache Camel and hawtio on OpenShift by James Strachan showing how to get started using JBoss Fuse 6.1 Early Access release on OpenShift for creating integration solutions based on Apache Camel in the hybrid cloud (via OpenShift Online for the public cloud or OpenShift Enterprise for on premise, or a combination of both).
  • CamelOne keynote 2013 by James Strachan showing the create, view, edit debug and provision of Camel routes using Fuse Fabric with the hawtio
  • demo of FMC and hawtio with Fuse Fabric
